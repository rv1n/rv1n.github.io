# API Московской биржи: бесплатные и платные, в чём разница

Краткий обзор каналов получения данных Мосбиржи и условий доступа.

---

## 1. ISS (Информационно-статистический сервер) — основной бесплатный API

**Что это:** HTTP API для доступа к котировкам, сделкам, истории, справочникам по всем рынкам (акции, облигации, валюта, срочный рынок, индексы).

**Документация:** [Программный интерфейс к ИСС](https://www.moex.com/a2193), [Справка ISS](https://iss.moex.com/iss/reference/).

### Бесплатный доступ (без договора)

- **Задержка:** данные могут предоставляться **с задержкой** (на практике часто доступны почти в реальном времени; официально биржа может давать с задержкой, в т.ч. до 15 минут).
- **Без регистрации:** запросы по HTTP, без логина и ключей.
- **Ограничения:** использование «только для ознакомления». Коммерческое использование, перепродажа данных и извлечение прибыли требуют договора с ПАО Московская биржа.

**Что можно получить через ISS бесплатно:**
- статика по инструментам и рынкам;
- котировки (marketdata: LAST, MARKETPRICE, WAPRICE и др.);
- история (свечи, тиковые данные);
- сделки (анонимные);
- метаданные, справочники.

**ДВС (дополнительная вечерняя сессия):** на рынке акций вечерняя сессия — примерно 18:50–23:50 МСК. Данные по торгам на той же площадке (TQBR и др.) в ISS приходят в общем потоке: LAST и прочие поля marketdata учитывают сделки и основной, и вечерней сессии. Индекс **IMOEX2** в ISS как раз считается по всем сессиям (включая вечернюю). Итого: **в ISS данные ДВС есть** — и по бумагам (через общий marketdata), и по индексу IMOEX2.

### Платный доступ к ISS (реальное время)

- **Режим реального времени** по тому же ISS — по **платной подписке** (договор с биржей).
- Условия и тарифы уточняются у Мосбиржи (отдел продаж данных).

**Итог по ISS:** бесплатно — те же методы и форматы, но с возможной задержкой; платно — гарантированное реальное время по тому же API.

## 2. Платные подписки на рыночные данные (Market Data)

**Что дают:** официальный доступ к данным в **реальном времени** по выбранным рынкам (не привязан только к ISS; может использоваться и для других каналов).

**Тарифы (ориентировочно, уточнять на сайте Мосбиржи):**
- Подписка на рынки (акции, облигации, валюта, фьючерсы/опционы, РЕПО): порядка **$90–100 USD/месяц** за точку доступа.
- Скидки при подписке на 4–6 и 7–12 месяцев.

**Оформление:** [Подписка на рыночные данные](https://www.moex.com/en/orders?realtime).

---

## 3. FAST — поток данных в реальном времени (UDP)

**Что это:** низколатентная доставка рыночных данных по протоколу **UDP multicast** (FIX/FAST) напрямую от торговых движков биржи.

**Особенности:**
- Задержка в среднем **менее 300 микросекунд**.
- Все рынки: акции, облигации, валюта, производные.
- Таблицы инструментов, котировки, сделки, индексы, заявки (в т.ч. полный лог заявок за отдельную плату).

**Тарификация (ориентировочно):**
- **FAST Gate (ПО):** регистрация порядка 2 000 ₽ + около 2 000 ₽/мес по каждому рынку (акции, валюта, производные — отдельно).
- **Полный лог заявок (Full orders log):** порядка 14 000 ₽/мес.

**Требования:**
- Выделенный канал связи через авторизованного провайдера Мосбиржи.
- Пропускная способность: от ~4 Мбит/с (срочный рынок) до ~15 Мбит/с (акции/валюта).

**Для кого:** профессиональные участники, алготрейдинг, HFT.

**Ссылки:** [FAST — real time market data](https://www.moex.com/a1528), [Конфигурация FAST](https://www.moex.com/a8533).

---

## 4. SIMBA

**Что это:** сервис данных по **срочному рынку** (фьючерсы, опционы) в реальном времени. Протокол FIX с кодированием SBE (Simple Binary Encoding).

**Для кого:** низколатентная торговля и аналитика по производным. Условия — по договору с биржей.

**Ссылка:** [Интерфейсы Мосбиржи](https://www.moex.com/a7939).

---

## 5. Шлюзовое подключение (технический доступ)

**Что это:** платное подключение к биржевой инфраструктуре (шлюзы для торговли и/или данных).

**Тарифы:** единовременные и ежемесячные платежи в зависимости от типа шлюза (порядка **4 000 – 30 000 ₽/мес** и выше). Точные цифры — на сайте Мосбиржи.

**Ссылки:** [Подключение к бирже](https://www.moex.com/s329), [Тарифы на подключение](https://www.moex.com/s655).

---

## 6. FTP и файловые данные

**Что есть:** архивные и справочные данные (в т.ч. по индексам, iNAV) в виде файлов.

**Пример:** `ftp.moex.com/pub/ClientsAPI/WebAPI/iNav_Indices/` — описание и схемы по индексам iNAV. Не поток в реальном времени, а справочники/архив.

**Использование:** коммерческое использование может требовать договора с биржей.
